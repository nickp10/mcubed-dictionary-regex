Dictionary
Dictionary:

Determines which dictionary is being used to see which words match the given criteria.

Loaded Dictionary:

States the current loaded dictionary as well as whether or not it has been loaded.
Search Criteria
Available Letters:

Specifies which letters are available to create a word from with a question mark ("?") specifying a wild card or blank tile.
Match Pattern:

Specifies the regex pattern that will be used in combination with the available letters to determine matching words.
* Word scores do NOT take into calculation: blank tile scores, letter multipliers, or word multipliers.
